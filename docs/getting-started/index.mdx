Diese Anleitung führt dich Schritt für Schritt durch die Einrichtung deines ALARMiator Servers.

## 📋 Voraussetzungen

<InfoBox type="info" title="Was du brauchst">
Bevor du startest, stelle sicher dass du:
- Ein Raspberry Pi 4 oder 5 oder ein Linux-Server steht zur Verfügung. Eine angemietete VPS ist auch eine sehr gute Option.
- Eine **stabile Internetverbindung** verfügbar ist
- Grundkenntnisse in **SSH und Linux** mitbringst (hilfreich, aber nicht zwingend)

→ [Detaillierte Systemanforderungen](/docs/systemanforderungen)
</InfoBox>

## 🎯 Installationsschritte

<StepByStep>
  <Step title="Hardware vorbereiten">
    - [Raspberry Pi / Linux Server einrichten](/docs/installationraspberrypilinuxserver)
    - Betriebssystem installieren
    - Netzwerkverbindung herstellen
  </Step>

  <Step title="ALARMiator Server installieren">
    <DocCardGrid>
      <DocCard
        title="Docker Compose (empfohlen)"
        description="Einfache Installation und Updates"
        to="/docs/installation-des-servers-via-docker-compose"
        icon="🐳"
      />
      <DocCard
        title="Native Installation"
        description="Nur für erfahrene Nutzer!"
        to="/docs/installationalarmiator-servernativ"
        icon="⚙️"
      />
    </DocCardGrid>
  </Step>

  <Step title="Netzwerk & SSL einrichten">
    1. [Eigene Domain einrichten](/docs/eigenedomaineinrichten)
    2. [SSL-Zertifikate konfigurieren](installation-des-servers-via-docker-compose#ssl-zertifikate-installieren)
    3. [Nginx Proxy Manager absichern](nginx-proxy-manager-verschluesseln) (für VPS)
  </Step>

  <Step title="Grundeinstellungen vornehmen">
    - [Erste Einstellungen nach Installation](erste-einstellungen-im-adminpanel-nach-der-installation)
    - [Adminbenachrichtigungen konfigurieren](einstellungen-admin)
    - [Berechtigungsgruppen anlegen](/docs/berechtigungsgruppen-anlegen)
  </Step>

  <Step title="Stammdaten anlegen">
    Jetzt legst du die organisatorischen Daten an:

    [Übersicht Stammdaten](/docs/uebersicht-stammdatenanlegen)

    Organisation → Standort → Gruppen → Mitglieder → Benutzer
  </Step>

  <Step title="Alarmierung aktivieren">
    Richte die Alarmierung ein:

    - [Übersicht Alarmierung](uebersicht-alarmierungen)
    - [Alarmwege und -profile konfigurieren](alarmwegematrixeinstellen)
    - [KatSys](katsyscloudconnector) oder andere Alarmquelle anbinden
  </Step>

  <Step title="Wallboard einrichten (optional)">
    Optionaler Schritt für Alarmmonitore:

    - [Übersicht Wallboard](/docs/uebersicht-wallboard)
    - Raspberry Pi Zero als Wallboard-Client einrichten
    - Widgets konfigurieren
  </Step>

  <Step title="Spaceport Client (optional)">
    Für abgesetzte Standorte:

    - [Übersicht Spaceport](/docs/uebersicht-spaceport)
    - Abgesetzte Standorte anbinden
  </Step>
</StepByStep>
